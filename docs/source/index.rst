Welcome to Ocean Carbon's documentation!
===================================

**Ocean Carbon Benchmarking** is an open-source, Python-based multimodel validation tool that can be used to evaluate the performance of ocean biogeochemical models. 

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
